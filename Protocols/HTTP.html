HTTP

HTTP stands for HyperText Transfer Protocol. HTTP runs on TCP port 80 and all web browsers run HTTP by default. There's also HTTPS in which the S is for Secure. HTTPS is more secure than HTTP because it encrypts everything that gets sent out. HTTP is the set of rules for transferring files (text, images, sound, video, and other multimedia files) on the World Wide Web. This is the protocol that you use when you type in the address of a website or click on a hyperlink on a webpage.

Whenever a user types in the URL (Uniform Resource Locator...aka the website's address) or clik on a link or video, your web browser builds an HTTP request and sends it to the website's server. The server builds a reply packet which is also HTTP and sends that back to you. You and the website's server sends HTTP packets back and forth.
