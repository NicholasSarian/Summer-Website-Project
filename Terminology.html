Terminology About

RAM

RAM stands for Random Access Memory. It's the memory that your computer (and other technological devices) use to run things like apps and processesses that must run in order to keep your device up and running. RAM is volatile (meaning the information it's storing will get lost if you power down the device) and is basically temporary memory.

BIOS

BIOS stands for Basic Input Output System. BIOS is a set of instructions for your device when it boots up. When you power on your device, it uses BIOS which tells it what to load, search, check, and find. One thing that the BIOS does is initiate the Bootloader which is responsible for finding the Operating System (OS).

Microprocessor

A Microprocessor is a physical component inside your computer and it's an integrated circuit that contains all the functions of a CPU (Central Processing Unit) of a computer. The microprocessor is the component that performs the instructions and tasks involved in computer processing. This component executes and manages the logical instructions that gets passed to it.

LAN

LAN stands for Local Area Network. This is a network of devices that are connected together that spans a building floor, several floors, or an entire building. LANs can span accross other buildings as long as the other buildings are within close proximity of each other. A LAN will consist of hosts (users), servers, switches, and routers.

WAN

A WAN (Wide Area Network) consists of multiple LANs and spans accross wide areas. A LAN in Lancaster, PA that's connected to a LAN in Philadelphia, PA is a good example of a WAN. WANs can even spread accross much wider distances, such as a LAN in New York that's connected to a LAN in California.

VLAN

A VLAN (Virtural LAN) exists on a switch and basically splits the switch into 2 or more separeate switches. A switch containing 2 VLANs will act as 2 switches. VLANs are useful for splitting up a large network into smaller sections. This helps a lot for convergence (when a change in the network has occured, the routers must get new updates. If a large network is split into smaller networks, the time it takes the routers to update the network topology will be significantly less).
